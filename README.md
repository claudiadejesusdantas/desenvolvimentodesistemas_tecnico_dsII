  <h2  style="color:red; text-align:center">Agenda 1</h2>
  <h3> Resumo </h3>
<ul>
  <li>
    A linguagem <strong>PHP (Hypertext Preprocessor – Pré processador de hipertexto)</strong> uma das mais conhecidas e utilizadas devido a sua tipagem dinâmica, utilizando o conceito server-side como alicerce. 
  </li>
  <li>O arquivo PHP é escrito entre os delimitadores 
  <code> 
    < ? php 
    ?> 
  </code> precisamos deles para identificar para o servidor que os códigos são PHP, o mesmo código pode possuir HTML.
  </li>
  <li>
    O comando echo em PHP tem a função de mostrar dados na tela do usuário.     
  </li>
  <li>
    O responsável pela interpretação dos códigos em PHP é o Web Server(Servidor Web).  
  </li>
  <li>
       O servidor gerencia as requisições dos clientes.
  </li>
  <li>
    O lado visível para o cliente é o front-end, ou client-side.
  </li>
  <li>
    O lado dos cálculos no servidor é o back-end, ou server-side, que precisa de um servidor web para trabalhar o servidor que iremos trabalhar está logo abaixo...
  </li>
</div>

<div id='usbwebserver8' >
  <h2  style="color:red; text-align:center"> USB Web Server 8 </h2>
  <br>
A ferramenta utilizada no módulo para ver o código funcionando será o <a href="https://usbwebserver.yura.mk.ua/" target="_blank"> USBWebServer8 </a> onde não precisa de instalação na máquina, é como um pendrive virtual.

<h3 style="color:red; text-align:center"> Instruções </h3>

* Realize o download da pasta compactada, extraia os arquivos.
* Instale e escolha a linguagem.
* Nas guias Apache e Mysql deverão aparece duas bolinhas verdes indicando o funcionamento adequado dos programas.


<h3 style="color:red; text-align:center"> Informações Importantes: </h3>

* Todos os arquivos deverão ser salvos na pasta **root**.
*  Nunca utilize caracteres especiais e espaços em branco para nomear arquivos.
* Abra o arquivo no seu navegador clicando em ```localhost``` e em seguida coloque o caminho, exemplo: ```localhost/Agenda5/ola.php```.
